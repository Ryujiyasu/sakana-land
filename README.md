# 🐟 さかなランド

3歳向けのさかな学習ゲームアプリです。車の中でも楽しく遊べます！

## ゲーム一覧

| ゲーム | 内容 |
|--------|------|
| 🎣 つりゲーム | さかなをタップして5ひきつろう |
| 🔢 かずゲーム | さかなが何匹いるか数えよう |
| 🧠 きおくゲーム | 同じさかなのカードを探そう |
| 🎨 いろゲーム | さかなが何色か答えよう |
| 📖 なまえゲーム | さかなの名前を覚えてクイズに挑戦！ |

## セットアップ

```bash
# 依存関係をインストール
npm install

# 開発サーバーを起動
npm run dev

# ビルド（本番用）
npm run build
```

## 必要環境

- Node.js 18以上
- npm 9以上

## 技術スタック

- React 18
- Vite 5
- Web Audio API（効果音）
- 外部ライブラリなし
